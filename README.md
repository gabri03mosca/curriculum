# Curriculum digitale - Gabriele Moscati

Versione più professionale del portfolio, pensata come curriculum tecnico per un profilo junior in Ingegneria di Internet.

## Struttura

- `index.html`
- `styles.css`
- `script.js`

## Come aggiornare il sito su GitHub

Carica questi tre file direttamente nella root del repository `curriculum`, sostituendo quelli vecchi.

La struttura corretta deve essere:

```text
curriculum/
├── index.html
├── styles.css
└── script.js
```

Poi attendi qualche minuto e apri:

```text
https://gabri03mosca.github.io/curriculum/
```

## Cose da personalizzare

Nel file `index.html` aggiorna:

- link LinkedIn
- eventuali link reali ai repository GitHub dei progetti
- eventuale foto profilo, se vuoi inserirla

## Nota di stile

Questa versione evita di elencare ogni esame come se fosse una lista scolastica.
Gli esami sono sintetizzati in aree tecniche, più adatte a un CV di ingegneria.
